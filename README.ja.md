# TwD3 (Japanese)/([English](README.md)) [![Chrome Web Store](https://img.shields.io/chrome-web-store/v/pbmcgkecchhocbndaicehjpdocoflbeg)](https://chrome.google.com/webstore/detail/twd3/pbmcgkecchhocbndaicehjpdocoflbeg)

TwD3は [TweetDeck](https://tweetdeck.twitter.com/) の２窓を検知するGoogle chrome拡張機能です。
  
# 紹介サイト[Japanese] 
https://atria64.github.io/twd3-web/
   
# 機能
- 検知時自動ウィンドウ遷移
  - 自動で余計なウィンドウを閉じ、既に開かれているTweetDeckウィンドウへ遷移します。
- クイックアクセス
  - アイコンを左クリックすることでTweetDeckへ素早くアクセスすることができます。
- TweetDeckブロックモード
  - 集中しなければならないときなど、TweetDeckを開きたくない場面でご利用ください。
